# OmniVision AI Studio

OmniVision AI Studio is an AI-powered system for image analysis and medicine verification.

## Features
- CNN-based animal image classification
- YOLO object detection
- OCR text extraction
- Text translation
- Medicine expiry date detection
- Explainable AI (Grad-CAM visualization)
- Live webcam object detection

## Technologies Used
- Python
- PyTorch
- YOLOv8
- EasyOCR
- Gradio
- OpenCV

## How to Run
1. Install dependencies
2. Run the Python file
3. Open the Gradio interface
